#SF Club Events

###Overview:
App clone of a Nightlife website where one could:

*Upload events	
*Sign up for a user account		
*Save favorited events.		
*Access DJs on soundcloud	

###Built with:
*Bootstrap	
*Javascript		
*Node.js	
*Express	
*Sequelize	
*Soundcloud API	(Generate your own Client ID to use)	
*Passport
